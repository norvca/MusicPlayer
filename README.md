# MusicPlayer
一个用原生 JS 写的网页音乐播放器

## 项目特点
- 基于屏幕宽高自适应
- 各区域功能组件化处理
- 对于数据请求采用 ajax 技术
- 歌曲源来自百度API，页面图标来自[iconfont](http://www.iconfont.cn/)

## 技术栈
ES5 + ajax + CSS3
